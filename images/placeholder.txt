## Especificaciones técnicas:

### Formatos recomendados:
- **Logo**: PNG con fondo transparente
- **Fotos**: JPG optimizadas para web
- **Iconos**: SVG cuando sea posible

### Tamaños recomendados:
- **Logo**: 200x80px (máximo)
- **Hero**: 800x600px
- **About**: 600x400px
- **Testimonios**: 100x100px (cuadradas)

### Optimización:
- Comprimir imágenes para web (calidad 80-85%)
- Usar formatos WebP cuando sea posible para mejor rendimiento
- Mantener tamaños de archivo menores a 500KB por imagen
